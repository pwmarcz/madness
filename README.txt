=====
MADNESS
=====

is a seven-day roguelike game. It's a simple dungeon crawler with themes of darkness and insanity.

A quick guide is available after pressing ? in the game.

-----
HOW TO RUN
-----

Under Linux:

  python madness.py

You need Python 2.6 and SDL.

Under Windows:

  madness.exe

If you get a "MSVCR90.dll not found" error, you need to download the Microsoft Visual C++ 2005 Redistributable Package (2.6 MB). This is a Python runtime's dependency. 

-----
TECHNOLOGY
-----

The game is written in Python 2.6 and uses the Doryen Library (libtcod - http://doryen.eptalys.net/libtcod/) for rendering, field-of-vision and pathfinding.

Windows binary was created using py2exe (http://www.py2exe.org/).
